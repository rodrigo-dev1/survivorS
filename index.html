Fluxo da mecânica

Como o sistema vai controlar o jogo.

1. Cadastre os jogadores reais.
2. Distribua as cartas fisicamente para cada pessoa.
3. Na primeira noite, o mestre vincula no sistema qual carta/função cada jogador recebeu.
4. O mestre registra as ações da noite: matar, salvar, silenciar, investigar, converter etc.
5. O sistema resolve conflitos e gera dois textos: relatório público para a cidade e relatório secreto para o mestre.
